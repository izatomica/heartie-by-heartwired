# Design System

This directory contains the complete design system specifications for Heartie, including colors, typography, spacing, and decorative elements.

## Design Personality

**Warm, organic, calm, human-first, slightly whimsical but professional.**

Think: a supportive friend who happens to be a marketing expert.

## Key Principles

- **Warm and organic** - No harsh angles, everything rounded and soft
- **Calm and supportive** - Colors are muted and harmonious
- **Human-first** - Hand-drawn elements, imperfect curves
- **Professional but approachable** - Clean but never cold

## Quick Reference

### Primary Colors
- **Burgundy**: `#7A2D4D` - Primary buttons, CTAs
- **Dark Teal**: `#1B6B6B` - Active states, progress
- **Dusty Pink**: `#D07080` - Highlights, current indicators
- **Cream**: `#FCF7F1` - Main background (NOT white)

### Funnel Colors
- **Awareness**: `#A8D5E5` - Sky Blue
- **Consideration**: `#9DCDB5` - Sage/Mint
- **Conversion**: `#E8C86B` - Gold/Mustard
- **Retention**: `#C5C0E8` - Lavender

### Typography
- **Headlines**: Bricolage Grotesque, 600-700 weight
- **Body**: Open Sans, 400-600 weight

### Key Measurements
- **Spacing base**: 8px
- **Card radius**: 16px
- **Button radius**: 10px
- **Common spacing**: 24px (space-6), 32px (space-8)

## Critical Rules

1. **Never use pure white backgrounds** - Always use Cream (#FCF7F1)
2. **All corners must be rounded** - No sharp angles anywhere
3. **Funnel colors are ONLY for activities** - Not for general UI
4. **Use Heartwired brand colors** - Do not substitute
5. **Maintain warm shadows** - Always use rgba(26, 26, 26, ...) with warm tint
