# Feedback Elements

## Toast Notifications

**Position:** Top-right, 24px from edges

**Toast:**
```css
Background: white
Border-radius: var(--radius-md)
Shadow: var(--shadow-lg)
Padding: 16px
Border-left: 4px solid [status color]
Max-width: 360px
Auto-dismiss: 5 seconds
```

**Status colors:**
- Success: var(--success)
- Warning: var(--warning)
- Error: var(--error)
- Info: var(--info)
